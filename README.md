# Simple Console Calculator using C


## Installation
1. git clone https://github.com/mboy1011/C_ConsoleCalc.git
2. cd C_ConsoleCalc
3. gcc -o Calc Calc.c
4. ./Calc


### Requirements
1. gcc (Compiler)
	*	Windows [Scoop](https://scoop.sh)
	*	Linux : <code>apt-get install gcc</code>
	*	Mac : <code>brew install gcc</code>
2. IDE (CodeBlocks)
